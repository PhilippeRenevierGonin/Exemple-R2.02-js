# Exemple-R2.02-js

pour lancer le serveur, il faut maven :
```
cd serveurjava
mvn clean package
mvn exec:java
```

pour lancer le client java :
```
cd client-chrono-java
mvn clean package
mvn javafx:run
```

(il faut se connecter puis lancer/stopper)

pour les pages webs, il faut ouvrir memory-mvc-reseau.html (ou la version pac)
